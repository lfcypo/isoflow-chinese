## 关键特性

- **高度可视化的拖放界面:** 用图标、区域和连接器来表达你的架构。
- **可定制:** 轻松扩展Isoflow与您自己的图标。
- **导出选项:** 将图表导出为代码或图像。

## 快速上手

安装 Isoflow [npm package](https://www.npmjs.com/package/isoflow) 和 isopacks(图标包):

- `npm install isoflow`
- `npm install @isoflow/isopacks`

欲了解更多信息，请参阅我们的 [在线文档](https://v2.isoflow.io/docs).
